# QuickSight Analysis Generator

Programmatic AWS QuickSight analysis generator for financial reporting.
Ships two independent dashboards sharing one theme, account, datasource,
and CLI surface.

## Dashboards

- **Payment Reconciliation** — sales, settlements, payments, and
  external-system matching for a merchant bank.
- **Account Reconciliation** — ledger/sub-ledger balances, transfers,
  and double-entry posting integrity for a community bank.

## Training Scenarios

Walk through realistic financial operations stories using the demo
dashboards:

- [Account Structure](Training_Story.md) — how the banks and accounts
  relate to each other.
