# For Your Role

Role-oriented entry points — "I'm a treasury operator, where do I
start?" Each page narrows the full handbook to the sheets, datasets,
and walkthroughs that role touches day-to-day, with a one-screen
"your view" diagram (added in O.1.c).

The intended user is someone landing on the docs without a clear sense
of which section to read first. Pick your role; the page funnels you
into the right [Reference](../reference/index.md) and
[Walkthroughs](../walkthroughs/index.md) entry points.

## Roles

Phase O.1.f populates this section. Until then the role-orientation
material lives in `training/handbook/for-*/` (extract via
`quicksight-gen export training`).

The planned set:

- For the operator — the day-in-the-life on the L1 dashboard.
- For the integrator — the L2 SPEC + L2 Flow Tracing validation flow.
- For the ETL engineer — the two base tables, the metadata catalog,
  and the matview refresh contract.
- For the executive — the Money Moved + Transaction Volume scorecard
  flow on the Executives app.
- For the compliance analyst — the Investigation question-shapes.
