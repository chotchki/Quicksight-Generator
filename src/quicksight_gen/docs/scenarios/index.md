# Scenarios

End-to-end stories grounded in the demo data — pick a scenario, follow
the breadcrumb across sheets, see how the dashboards surface the same
underlying event from operator, integrator, executive, and compliance
perspectives.

The intended user is anyone evaluating the dashboards: "show me what
this looks like in practice." Each scenario uses Sasquatch National
Bank (the demo persona) so the rows match what `demo apply` plants.
